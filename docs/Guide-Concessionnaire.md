# Guide concessionnaire — RideCloudMoto

Ce guide explique comment utiliser le **portail concessionnaire** pour offrir RideCloudMoto à vos clients, suivre leurs licences et les fidéliser.

---

## 1. En bref

À chaque vente d'un deux-roues, vous offrez à votre client **12 mois d'accès gratuit** à RideCloudMoto (le carnet d'entretien intelligent de sa moto/scooter). Aucun code promo à gérer : **l'immatriculation du véhicule sert de code d'activation**.

- Le client s'inscrit, saisit son immatriculation, et son offre démarre.
- Il est automatiquement **rattaché à votre concession** (il voit vos coordonnées, vos horaires, vos promotions et peut prendre rendez-vous).
- Vous pouvez **prolonger** son accès (par exemple après une révision) pour l'inciter à revenir à l'atelier.

---

## 2. Votre compte

Vous accédez au portail avec un **compte e-mail dédié** créé par l'équipe RideCloudMoto.

1. Vous recevez une **invitation par e-mail** (ou vous créez un compte avec l'e-mail communiqué).
2. Vous définissez votre mot de passe.
3. À la connexion, vous arrivez directement sur votre **portail** (`/portail`).

> Un compte = une concession. Deux rôles existent : **Responsable** (owner) et **Vendeur** (staff). Les deux ont accès aux mêmes fonctions du portail.

---

## 3. Le portail (`/portail`)

Trois espaces, accessibles depuis la barre de navigation « Espace pro » :

### Tableau de bord
Vue d'ensemble : **ventes enregistrées**, **licences activées** (clients ayant saisi leur code) et **fiches en attente** d'activation.

### Nouvelle vente
Le cœur de votre activité quotidienne.

### Licences
La liste de vos clients, avec la possibilité de **prolonger** une licence.

---

## 4. Enregistrer une vente

À la livraison d'un véhicule :

1. Ouvrez **Nouvelle vente**.
2. Saisissez l'**immatriculation** au format SIV `AB-123-CD` (**obligatoire**) — c'est le code d'activation du client.
3. Complétez (facultatif mais recommandé) : **modèle**, **date d'achat**, **coordonnées du client** (prénom, nom, e-mail, téléphone).
4. Cliquez sur **Enregistrer la vente**.

Un message de confirmation affiche le code. Remettez-le au client, par exemple :
- via le **flyer** avec QR code (voir §7),
- ou simplement en lui indiquant qu'il devra saisir son **immatriculation**.

> Le client n'a **rien à payer** pendant toute la durée de l'offre. L'offre n'est pas automatique : le client doit s'inscrire et saisir son code une fois.

---

## 5. Suivre et prolonger les licences

Dans **Licences**, vous retrouvez toutes vos ventes :

| Statut | Signification |
|---|---|
| **En attente** | Le code a été créé, le client ne l'a pas encore activé |
| **Activée** | Le client a créé son compte et démarré son offre |

Filtres disponibles : *Toutes*, *Activées*, *En attente*.

### Prolonger (fidélisation)
Pour une licence **activée**, le bouton **Prolonger** ajoute une nouvelle période (la durée configurée pour votre concession) :

- si l'offre est encore active → la période s'ajoute à la date de fin ;
- si l'offre a expiré → elle repart d'aujourd'hui.

**Cas d'usage** : offrir une année supplémentaire après une révision à l'atelier, pour encourager le client à revenir.

---

## 6. Votre vitrine dans l'app (carte « Mon concessionnaire »)

Chaque client rattaché à votre concession voit une page **Mon concessionnaire** dans l'app, qui peut afficher :

- votre **logo** et vos **couleurs** (la couleur de l'app s'adapte à votre marque) ;
- votre **adresse**, **téléphone**, **e-mail**, **site web** ;
- un bouton **Itinéraire GPS** ;
- un bouton **Prendre rendez-vous** ;
- vos **promotions** en cours ;
- vos **horaires** d'ouverture.

> La configuration de ces éléments (logo, couleurs, coordonnées, horaires, promotions, durée d'offre) est réalisée par l'équipe RideCloudMoto lors de la mise en place de votre concession. Communiquez-nous vos éléments pour les mettre à jour.

---

## 7. Le flyer avec QR code

Pour faciliter l'inscription du client :

- **Flyer prêt à l'emploi** : ouvrez `/flyer.html`, renseignez (à l'écran) l'immatriculation ou laissez l'URL d'inscription, le **QR code se met à jour**, puis imprimez.
- **Flyer vierge** : `/flyer-concessionnaire.html` (zone code à remplir au stylo).

Le client scanne le QR → arrive sur la page d'inscription → crée son compte → saisit son immatriculation → c'est prêt.

---

## 8. Questions fréquentes

**Le client a perdu son code ?**
Son code est son **immatriculation**. Il la ressaisit dans *Paramètres → activer un code*.

**Le client change de véhicule ?**
Enregistrez une nouvelle vente avec la nouvelle immatriculation.

**Que se passe-t-il à la fin de l'offre ?**
L'app passe en **lecture seule** : le client garde l'accès à son historique, ses factures et ses exports, mais ne peut plus rien ajouter ni modifier. Il peut souscrire au **Premium** pour réactiver l'écriture, ou vous pouvez **prolonger** sa licence.

**Puis-je voir les données d'entretien de mes clients ?**
Non. Le portail vous montre l'état des **licences** (activée / en attente) et les coordonnées saisies à la vente, mais pas le contenu privé du carnet d'entretien du client.

**Comment ajouter un collègue à mon portail ?**
Contactez l'équipe RideCloudMoto : elle rattache le compte (par e-mail) à votre concession.
